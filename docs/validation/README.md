# Ontology Validation Results

This directory contains validation test results from manual ontology population against real published documents.

See: `plans/ontology-validation-test.md` for the test plan.

## Contents

- `sample-selection.md` - Which documents were tested and why
- `doc-*.md` - Per-document findings (nodes, relationships, gaps)
- `synthesis.md` - Cross-document analysis and patterns
- `recommendations.md` - Actionable next steps
